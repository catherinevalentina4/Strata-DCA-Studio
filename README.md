# Strata-DCA-Studio
📉 StrataDCA — throw in your well's production data and watch it auto-detect the decline, fit Arps curves (Exponential/Harmonic/Hyperbolic), and spit out P10/P50/P90 EUR forecasts via Monte Carlo magic. No backend, no install — just open the HTML and go. Built for engineers who like their curves smart and their tools simple.
